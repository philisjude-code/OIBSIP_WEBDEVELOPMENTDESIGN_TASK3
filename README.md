# Oasis Infobyte Web Development Internship

## Task 3 – Temperature Converter

### Objective
The objective of this project was to develop an interactive temperature converter that 
accurately converts temperature values between Celsius, Fahrenheit, and Kelvin using JavaScript. 
The project focuses on implementing input validation, mathematical conversions, and responsive user interface design.

### Steps Performed
- Designed a simple and responsive user interface for temperature conversion.
- Implemented input fields for entering temperature values and selecting conversion units.
- Developed conversion logic for Celsius, Fahrenheit, and Kelvin using JavaScript.
- Added input validation to handle invalid or empty values.
- Displayed accurate converted results instantly based on user input.
- Styled the application using CSS to provide a clean and user-friendly experience.

### Tools Used
- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- Git & GitHub

### Outcome
Successfully developed a responsive temperature conversion application capable of performing 
accurate unit conversions with proper validation and error handling. This project strengthened my understanding 
of JavaScript functions, DOM manipulation, mathematical logic, and responsive web application development.
